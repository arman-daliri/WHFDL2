# Article
**WHFDL2: Leveraging a Unified Fitness Function for Reinforcement Learning-Based Feature Selection in Gastric Cancer Classification Using Metabolomics and Metaproteomics**

# How to Run
The implementation of the feature selection method is provided in the `WHFDL2.ipynb` notebook.

First, install the required packages by running:
```sh
pip install -r requirements.txt
```
Then, run the `WHFDL2.ipynb` notebook cell by cell.
